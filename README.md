#Generate Normal Map

Create a normal map with a png. The code will generate 100 NormalMap once executed. You can pick up which is best for you. 

## Required

- Python3
- Scipy
- Numpy
- imageio

## Usage

```
python3 GenerateNormalMap.py my.png
```
